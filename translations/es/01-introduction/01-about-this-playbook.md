# Acerca de Este Playbook

## Propósito de Este Playbook

> **Nota**: A medida que las sedes de BitDevs crecen, mejoran y comparten nuevas lecciones, este playbook evolucionará para reflejar esos aprendizajes y fortalecer el ecosistema.

El BitDevs Playbook de [Btrust](https://www.btrust.tech/) es una guía integral para organizadores, patrocinadores y socios involucrados en el crecimiento de comunidades técnicas enfocadas exclusivamente en Bitcoin en África. Proporciona un marco claro sobre cómo iniciar, operar y mantener sedes de BitDevs, al tiempo que describe los principios, expectativas y estándares que aseguran que cada sede funcione con la misma calidad, transparencia e impacto.

Este playbook sirve como:
- Un manual para organizadores que están creando o gestionando sedes de BitDevs.
- Una referencia para patrocinadores y socios para entender los procesos, reportes y estándares de rendición de cuentas.
- Un recurso para miembros de la comunidad que desean alinearse con el movimiento BitDevs más amplio.

## Cómo Usar Este Playbook

Este playbook está dividido en secciones claras que reflejan el ciclo de vida completo de una sede BitDevs, desde la creación, pasando por las operaciones, el control de calidad, y el eventual cierre (si es necesario). Si bien todos los involucrados deberían leer el playbook completo para entender cómo las sedes BitDevs operan como una red cohesiva, las secciones a continuación ofrecen una forma ágil de comenzar según tu rol:

- **Nuevos Organizadores:** Comienza con la Sección 2: ["Iniciar una Sede BitDevs"](../02-starting-a-bitdevs-location/01-setting-up-a-location.md) para aprender cómo crear una nueva sede desde cero. Continúa con la Sección 4: ["Control de Calidad"](../04-quality-control/01-core-standards.md) para establecer estándares desde el primer día. Luego, lee la Sección 6: ["Guía para Organizadores"](../06-guide-for-organizers/01-roles-and-responsibilities.md) para orientación sobre desarrollo de equipo y personal.
- **Sedes Activas:** Enfócate primero en la Sección 3: ["Operar una Sede BitDevs"](../03-running-a-bitdevs-location/01-meetup-planning-and-coordination.md) para orientación operativa. Revisa la Sección 4: ["Control de Calidad"](../04-quality-control/01-core-standards.md) regularmente. Consulta la Sección 6: ["Guía para Organizadores"](../06-guide-for-organizers/01-roles-and-responsibilities.md) para seguir desarrollando tu equipo y habilidades.
- **Patrocinadores y Colaboradores:** Consulta la Sección 7 ("Guía para Patrocinadores") para modelos de financiamiento, rendición de cuentas y expectativas de colaboración.
- **Todos:** Comienza con la Sección 1: ["Introducción"](01-about-this-playbook.md) para conocer los principios fundacionales, identidad y ética de BitDevs.
- **Anexos:** Contienen plantillas, ejemplos y listas de verificación referenciadas a lo largo del playbook para uso práctico.

## A Quién Está Dirigido Este Playbook

Este playbook está diseñado para todos los contribuyentes al ecosistema BitDevs:

- Organizadores de BitDevs: Personas que lideran o co-lideran sedes locales de BitDevs.
- Equipos Organizadores: Voluntarios que apoyan la logística, comunicación y contenido técnico.
- Miembros del Equipo Btrust: Personal de Btrust que coordina patrocinios, aseguramiento de calidad y reportes.
- Patrocinadores o Socios: Organizaciones o personas interesadas en apoyar iniciativas técnicas de base.
- Participantes y Desarrolladores: Asistentes a BitDevs y miembros de la comunidad que desean entender mejor cómo funcionan las sedes y cómo pueden contribuir.

Cualquier persona que planifique, financie, apoye o asista a meetups de BitDevs puede beneficiarse de revisar, entender y aplicar esta guía.
