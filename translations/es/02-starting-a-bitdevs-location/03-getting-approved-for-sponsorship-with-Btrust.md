# Obtener Aprobación para Patrocinio (con Btrust)

El patrocinio de Btrust permite a las sedes de BitDevs operar de manera fluida y sostenible. Cubre gastos elegibles como lugar, refrigerios y logística básica, e incluye apoyo estructurado para operaciones, control de calidad y reportes.

## Requisitos de Elegibilidad para Patrocinio

Para calificar para el patrocinio de Btrust, una sede de BitDevs debe:

- Comprometerse a ser exclusivamente Bitcoin y organizar discusiones técnicas enfocadas en Bitcoin.
- Tener al menos dos organizadores activos para asegurar la rendición de cuentas y la continuidad. Idealmente incluir organizadores con experiencia técnica e involucramiento comunitario (por ejemplo, egresados de los programas de [Btrust Builders](https://www.btrust.tech/builders), [Chaincode Labs](https://chaincode.com/) o [Bitshala](https://bitshala.org/)). En regiones con pocos desarrolladores de Bitcoin, un único organizador comprometido puede iniciar una sede. En tales casos, el organizador debe trabajar activamente para incorporar otro organizador dentro de un plazo definido (por ejemplo, seis meses) para apoyar la sostenibilidad y el crecimiento.
- Aceptar realizar meetups mensuales y mantener documentación adecuada.
- Seguir los principios fundamentales de BitDevs de neutralidad, privacidad y apertura.
- Enviar reportes mensuales de Meetup y Finanzas usando las plantillas estándar.
- Participar en las reuniones mensuales de organizadores para aprendizaje continuo y rendición de cuentas.

## Proceso de Solicitud

El proceso de solicitud de patrocinio de Btrust incluye los siguientes pasos:

1. **Envío de Solicitud**
   Completa el [formulario de solicitud](https://btrust.homerun.co/bitdevs-application-sponsorship/en) con los detalles del organizador y una propuesta breve describiendo tu contexto local, objetivos y actividades esperadas.

2. **Revisión y Aprobación**
   Las solicitudes se revisan de forma continua, típicamente dentro de dos a tres semanas.

3. **Llamada Introductoria**
   Los solicitantes aprobados tendrán una breve llamada introductoria con el equipo de Btrust para:
   - Revisar planes y actividades propuestas
   - Aclarar expectativas y requisitos de reportes
   - Confirmar la disposición para organizar un meetup

4. **Período de Prueba (Tres Meses)**
   Las sedes aprobadas son incorporadas a la red BitDevs, añadidas a los canales de comunicación compartidos y se les otorga acceso a herramientas operativas y patrocinio.
   Todas las sedes comienzan con un patrocinio de prueba de tres meses a partir de su primer evento. Durante este período, las sedes deben:
   - Enviar reportes mensuales de Meetup y Finanzas
   - Participar en la reunión mensual de organizadores
   - Seguir el formato Socrático y mantener la neutralidad

## Lo que Btrust Proporciona

El patrocinio de Btrust incluye:

- Asignación de fondos para meetups mensuales
- Plantillas estándar para planificación de meetups, finanzas y reportes
- Orientación operativa y acceso a la red global de organizadores BitDevs
- Retroalimentación continua, sesiones de aprendizaje y visibilidad para sedes activas

## Continuación del Patrocinio

El patrocinio es continuo pero depende de mantener el estatus activo de la sede, asistencia razonable, reportes oportunos y alineación continua con este playbook.

Este modelo también puede servir como referencia para otras sedes que buscan asociaciones estructuradas o diseñan sus propios marcos de patrocinio.
